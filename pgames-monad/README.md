# Monad Games and Zero-Sum Game Equilibrium Solving

## Requirements

* ocaml 4.0 or greater

* opam

* opam's extlib and ocaml-glpk packages:

```bash
opam install extlib ocaml-glpk
```

## Building

```bash
make
```

## Running

* Various examples are named exp_EX, and their source code is
  exp_EX.ml .

```bash
./exp_guess_my_number
```
